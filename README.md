## Usage

install dependency

```bash
$ npm i 
```

launch local server

```bash
$ npm run docs:dev
```

## Contents

Laravel を 利用してREST API を作成するフローを紹介します。

Laravel ならではの Database 操作や ルートの記述などを利用して
SPA アプリケーションなどで活用可能な REST API を構築してみましょう。

## こんな方におすすめ
